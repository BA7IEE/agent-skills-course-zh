# GitHub 首发整理

这份文件不是课程正文，而是为了帮你把仓库更顺手地发到 GitHub。

## 建议仓库名

- `agent-skills-course-zh`
- `agent-skills-cn`
- `agent-skills-chinese-course`

如果你希望更像“中文学习版课程仓库”，第一项最稳。

## 建议仓库简介

你可以直接选一条：

### 简洁版

把《Agent Skills with Anthropic》重写为中文学习路径的一套 GitHub 文档课程。

### 标准版

一套面向中文零基础 / 弱英文用户的 Agent Skills 文档课程，保留原课主线，补术语解释、中文语境说明与学习路径。

### 偏开源传播版

Chinese adaptation of Agent Skills learning materials for beginners, with rewritten lessons, glossary, FAQ, and structured study path.

## 首发仓库说明建议

首次公开时，你可以在仓库说明区强调这三点：

- 这是中文学习版，不是官方中文版
- 保留原课主线，但已按中文读者习惯重写
- 公开仓库不保留原始英文资料副本，只保留章节映射和中文整理内容

## 建议首发顺序

1. 先确认仓库名、简介和可见性设置
2. 上传当前目录内容并检查 README 首页显示效果
3. 设置 topics
4. 补一张封面图或社交分享图到 `assets/`，如果你打算做传播
5. 提交首发 commit
6. 如果需要，再补一个 `v0.1.0` 标签

## 建议首发版本说明

如果你要发一个 Release，可以直接参考下面这段：

### v0.1.0

首个公开版本，包含：

- 10 章中文课程正文
- 术语表、FAQ、课程映射与延伸阅读
- README、NOTICE、LICENSE、贡献说明与发布清单
- 面向中文零基础 / 弱英文用户的整体重写与课程化整理

说明：

- 本仓库为学习性改写与中文传播整理版本
- 不包含原始英文课程资料副本
- 涉及官方产品能力时，请以官方最新文档为准

## 发布前最后看一眼

- README 首页是否是你愿意公开代表自己的版本
- NOTICE 和 LICENSE 是否符合你的风险预期
- 仓库里是否已经没有不想公开的临时文件
- 是否要再补一张封面图
