# Contributing

欢迎继续完善这套中文课程。

## 适合贡献的方向

- 修正文案错误和错别字
- 优化中文表达，让内容更适合零基础用户
- 补术语解释、FAQ 和章节映射
- 增加不改变原课核心主线的辅助说明
- 增加更清晰的 README 导航和发布材料

## 不建议随意改动的部分

- 不要把课程主线改成完全不同的体系
- 不要把中文正文改成逐字翻译腔
- 不要随意删除课程映射关系
- 不要在未核查来源的情况下新增第三方受限材料

## 文风建议

- 先讲清楚“它解决什么问题”，再讲术语
- 尽量降低名词堆叠
- 面向中文零基础 / 弱英文用户写作
- 不要为了“专业”牺牲可读性

## 提交前建议自检

- 章节标题是否清晰
- 链接是否有效
- 是否破坏了课程映射关系
- 是否引入了新的版权或引用风险
- 是否和当前仓库定位一致

## 关于第三方材料

如果你的改动涉及：

- 外部截图
- 外部课程原文
- 第三方文档内容搬运

请先确认是否适合放入公开仓库，再决定是否提交。
