# 资料区说明

这里不是正文课程，而是整套课的辅助阅读区。

如果把 `lessons/` 看成主线内容，那 `resources/` 更像是给中文读者准备的配套工具箱：你不一定要一次读完，但在读课、复习、对照原课和继续延伸时，它会很有用。

## 建议怎么用

- 第一次学习：先按顺序读正文，遇到术语再回来看术语表和 FAQ
- 第二次复习：配合课程映射和延伸阅读，把整套课和原课、官方资料连起来
- 准备公开发布：重点看课程映射、NOTICE 和 LICENSE，确认引用边界

## 文件说明

- [course-map.md](course-map.md)：原课程与中文章节的映射关系，适合对照阅读
- [glossary.md](glossary.md)：核心术语表，适合第一次扫清概念障碍
- [faq.md](faq.md)：零基础用户最容易卡住的问题，适合补理解边界
- [references.md](references.md)：学完后继续往下查什么，适合延伸学习
