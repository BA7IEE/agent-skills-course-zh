# summarize-lecture

## 目标

把输入材料整理成便于继续学习和复习的结构化输出。

## 适用场景

- 课程逐字稿整理
- 文章提纲提取
- 会议记录归纳

## 推荐流程

1. 先识别输入材料的主题和结构。
2. 提取核心观点、关键术语和例子。
3. 按“主题 - 要点 - 说明”的形式整理输出。
4. 如果材料很长，先分段处理，再统一合并。

## 输出要求

- 保留原始逻辑顺序
- 不要凭空补充结论
- 尽量让结果适合继续加工成笔记或练习题

## 边界情况

- 输入材料过短时，直接说明信息不足
- 输入结构混乱时，先做主题归类，再输出提纲
